## SeqGAN
通过SeqGAN生成小黄鸡式的对话。细节请阅读[paper](https://arxiv.org/abs/1609.05473)

## 训练
```
python main.py
```

## 训练语料
训练语料来自[小黄鸡语料](https://github.com/fateleak/dgk_lost_conv/tree/master/results)
